Prosta gra konsolowa napisana w C++. Gracz zgaduje wylosowaną liczbę na jednym z trzech poziomów trudności: Noob (1–50), Medium (1–100) lub Pro (1–250). Program podpowiada, czy liczba jest za duża lub za mała, liczy liczbę prób oraz zapisuje i sortuje wyniki dla każdego poziomu.

Projekt edukacyjny stworzony w celu nauki podstaw C++, pracy z pętlami, tablicami, losowaniem liczb i obsługą menu tekstowego.
